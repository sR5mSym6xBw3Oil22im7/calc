# Calculator

A simple calculator web app split into three files: `index.html`, `style.css`, and `script.js`.

## How to use

1. Open `index.html` in your browser.
2. Use the buttons or keyboard to enter numbers and operations.
3. Press `=` or `Enter` to calculate.
4. Use `C` to clear and `⌫` to delete the last digit.

## Files

- `index.html` contains the calculator layout and links to the CSS and JavaScript.
- `style.css` contains the visual design and responsive styles.
- `script.js` contains the calculator behavior, button handling, keyboard support, and theme switching logic.

## Features

- Click or keyboard input support
- Basic arithmetic operations: addition, subtraction, multiplication, division
- Theme selection using the dropdown menu
- Responsive layout for smaller screens
